Ztock

Ztock is query tool that allows users to lookup stocks listed on US exchanges with a valid ticker symbol. The search utility allows users to obtain a realtime trading price as well as a company profile that includes key financial metrics for the company. Users can save their tickers to a personal profile. A user login is required to use the utility and can be easily created by providing a username and password.
